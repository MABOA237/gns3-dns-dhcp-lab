# GNS3 DNS & DHCP Lab

## Overview
This projectis a virtual network lab built using GNS3. It implements core network services such as DNS and DHCP in a simulated environment.

## Technologies Used
-GNS3
-Linux (Ubuntu server)/n
-Bind9 (DNS)
-kea DHCP (in progress)
-VLANs
-static Routing

## Current Progress
-Network topology created in GNS3
-VLAN configuration completed
-static routing completed
-DNS service implemented using Bind9

## Next Steps
